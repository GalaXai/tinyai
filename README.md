# tinyai

A tiny library for building and training neural networks using [tinygrad](https://github.com/tinygrad/tinygrad).

# Trainings - Examples
- [] [MNIST](./examples/mnist.py)


# Installation
```bash
git clone -b tinyMain https://github.com/GalaxUniv/tinyai.git
cd tinyai
pip3 install -e .
```
